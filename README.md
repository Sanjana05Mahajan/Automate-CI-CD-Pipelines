# Automate-CI-CD-Pipelines
Automate CI/CD Pipelines using Lambda. # 🚀 Automated CI/CD Pipeline using AWS Lambda & CodePipeline

## 📌 Overview
Serverless CI/CD automation project that triggers AWS CodePipeline using Lambda for faster deployments.

## 🧰 Tech Stack
- AWS Lambda
- AWS CodePipeline
- IAM
- Python (boto3)

## ⚙️ Workflow
Lambda Trigger → Pipeline Start → Source → Deploy

## ✨ Features
- Fully automated deployment trigger
- Serverless architecture
- IAM secure access
- Fast deployment pipeline

## 🚀 Setup Steps
1. Create Lambda
2. Add IAM Role
3. Create CodePipeline
4. Upload Code
5. Test Lambda
